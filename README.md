#### 주제 및 설명 : 코로나시대에 신입생들의 불편함을 덜어주는 앱
#### 기능 및 설명 : 기능 및 설명 
 ##### -> 코로나로 낯선 학교에 길을 잃지 않게 도와 줄 지도 기능
 ##### -> 코로나로인해 동기들끼리 흔한 술자리 한번 제대로 갖지 못하는 신입생들을 위해 신입생만 이용할 수 있는 채팅방
 ##### -> 안그래도 어려운 학교생활 선배들이 모으고 모은 유익한 팁과 유의점을 알려주는 안내판
 ##### -> 현제 코로나 상황을 한번에 알려줄 배너
 ##### -> 학교의 코로나 방침과 코로나 대응상황을 알려줄 공지사항
